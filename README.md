<h1 align="center">Конструктор тестов</h1>

## Description

Конструктор тестов. Технологии: BackEnd: NodeJs/ NestJS/ DB: MongoDB FrontEnd: Angular2

## Installation

```bash
$ npm install
$ cd client
$ npm install
```

## Running the app

```bash
#Back-END
# development (back-end)
$ npm run start

# watch mode (back-end)
$ npm run start:dev

# production mode (back-end)
$ npm run start:prod

#Front-END
# Development mode (Front-end)
$ npm run start

#Build mode (front-end)
$ npm run build

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
